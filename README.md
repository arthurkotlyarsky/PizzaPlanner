# Pizza Planner

Pizza Planner is a Java application designed to order pizza securely using AWS.

## Built With

* [Amazon RDS](https://aws.amazon.com/rds/) - Back-end database and security
* [Netbeans](https://netbeans.org/) - The IDE used
